# 🍕 Pizza Restaurant Website

A modern and responsive Pizza Restaurant landing page built using **HTML5** and **CSS3**. The website features an attractive hero section, menu showcase, image gallery, carousel, contact section, and footer inspired by a Figma design.

## 🚀 Features

* Responsive Navigation Bar
* Hero Section with Background Image
* About Us Section
* Menu Showcase
* Image Carousel
* Gallery Layout
* Find Us / Contact Section
* Footer with Contact Information
* Modern Dark Theme UI
* Fully Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts

## 📸 Sections Included

### Home

Beautiful hero banner with pizza background and promotional content.

### About Us

Information about the restaurant and its story.

### Menu

Displays various food items with images and descriptions.

### Carousel

Interactive image slider showcasing featured dishes.

### Gallery

Grid-based image gallery for restaurant food items.

### Find Us

Location information and contact details.

### Footer

Email and phone contact information.
